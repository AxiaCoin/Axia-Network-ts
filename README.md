# @axia-js/ts

TypeScript definitions for packages that are not (yet) available on [DefinitelyTyped](https://github.com/DefinitelyTyped/DefinitelyTyped) but are used in `@axia` projects.

In general, updates here should have corrsponding PRs on the DefinitelyTyped repo. Tracking should be provided for these.
